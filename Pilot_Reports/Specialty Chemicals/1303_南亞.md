# 1303 - [[南亞]]

## 業務簡介
**板塊:** Basic Materials
**產業:** Specialty Chemicals
**市值:** 642,397 百萬台幣
**企業價值:** 737,502 百萬台幣

南亞塑膠 (1303，Nan Ya Plastics) 為[[台塑集團]]四大公司之一，台灣最大塑膠加工及電子材料製造商，營收約 2,600 億元。四大事業群：(1) 塑膠加工 — [[PVC]]管材/膠布、[[BOPP]]薄膜、工程塑膠；(2) 化工 — [[環氧樹脂]]、[[雙酚A]]、可塑劑、[[乙二醇]]；(3) 電子材料 — [[銅箔基板]] (CCL，全球前三大)、[[玻纖布]] (全球前三大)、[[銅箔]]，供應[[欣興]]、[[臻鼎]]、[[華通]]、[[聯茂]]、[[台光電]]等 PCB 大廠，受惠[[AI 伺服器]]需求；(4) 聚酯纖維 — [[PET]]膜/加工絲。重要轉投資涵蓋子公司[[南亞電路板]] (南電，8046，全球前三大[[ABF 載板]]供應商，服務[[Intel]]、[[AMD]]、[[NVIDIA]]) 及[[南亞科技]] (2408，DRAM)。上游石化原料來自集團內[[台塑]]、[[台化]]、[[台塑石化]]。

## 供應鏈位置
**上游 (石化原料):**
- **集團內部:** [[台塑]] (乙烯)、[[台化]] (酚/丙酮)、[[台塑石化]] (丙烯/石腦油)

**中游 (南亞本體):**
- **塑膠:** [[PVC]]管材/膠布、[[BOPP]]薄膜、工程塑膠
- **化工:** [[環氧樹脂]]、[[雙酚A]]、可塑劑、[[乙二醇]]
- **電子材料:** [[銅箔基板]]/[[玻纖布]]/[[銅箔]] (全球前三大)
- **聚酯:** [[PET]]膜/加工絲

**下游 (終端):**
- **PCB 大廠:** [[欣興]]、[[臻鼎]]、[[華通]]、[[聯茂]]、[[台光電]] — CCL/玻纖布/銅箔
- **載板:** [[南亞電路板]] (子公司) — [[ABF 載板]]供[[Intel]]、[[AMD]]、[[NVIDIA]]
- **DRAM:** [[南亞科技]] (轉投資) — 記憶體
- **建材:** 國內營造業 — PVC 管材
