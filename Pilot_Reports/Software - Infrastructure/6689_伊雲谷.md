# 6689_伊雲谷

## 業務簡介
**板塊:** Technology
**產業:** Software - Infrastructure
**市值:** 4,848 百萬台幣
**企業價值:** 2,864 百萬台幣

伊雲谷數位科技股份有限公司 (6689) 是亞洲首家掛牌上市的雲端數位管理服務公司(MSP/MSSP)。伊雲谷專注為企業提供一站式數位轉型顧問與多雲託管服務。公司是[[Amazon]](AWS)在大中華區首家「核心級服務合作夥伴」，同時也整合[[Microsoft]](Azure)、[[Google]](GCP)與[[Oracle]]等主流公有雲。除基礎架構搬遷與維運，伊雲谷更提供大數據分析、AI應用、資訊安全託管與永續碳管理(ESG)等高附加價值服務，並自主研發雲端與AIoT管理平台，服務全球逾5,000家企業客戶。

## 供應鏈位置
*   **上游**: 全球三大公有雲服務平台商(AWS、Azure、GCP)、企業級軟體原廠(如SAP、[[Oracle]]、[[Salesforce]])。
*   **中游**: **伊雲谷** (多雲架構搬遷、企業雲端託管維運MSP、IT數位轉型諮詢與AI大數據/ESG加值應用服務)。
*   **下游**: 需要進行IT系統雲端化或數位轉型的各產業企業端客戶，涵蓋公部門、金融、電信、零售及製造業。

## 主要客戶及供應商
### 主要客戶
*   政府機關、國內外金融機構(如中華開發金控)、大型科技製造業(如微星)、連鎖電商平台及遊戲公司。

### 主要供應商
*   [[Amazon Web Services]](AWS)、[[Microsoft]]、[[Google Cloud]]、[[SAP]]等全球頂級雲端基礎設施與軟體巨擘。
