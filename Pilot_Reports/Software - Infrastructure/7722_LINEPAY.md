# 7722 - LINEPAY

## 業務簡介
**板塊:** Technology
**產業:** Software - Infrastructure
**市值:** 20,978 百萬台幣
**企業價值:** 11,419 百萬台幣

連加網路商業股份有限公司 (7722，LINE Pay Taiwan) 為 [[LINE]] 集團在台灣經營第三方支付與行動支付之子公司。LINE Pay 為台灣行動支付市場主要品牌之一，服務涵蓋餐飲、購物、飲料、生活、交通、娛樂及美妝旅遊七大消費場域。營收主要來自第三方支付交易手續費抽成，依據合作商家店面交易金額收取佣金。LINE Pay 透過 [[LINE]] 通訊平台龐大用戶基礎進行導流，並與多家銀行合作提供信用卡綁定與帳戶儲值功能。2024 年營收達 62.96 億元，年增 27.8%。

## 供應鏈位置
**上游 (金融與技術平台):**
- **母集團:** [[LINE]] — 通訊平台用戶導流與技術支援
- **收單銀行:** [[中國信託]]、[[國泰世華銀行]]等合作銀行
- **信用卡組織:** [[VISA]]、[[Mastercard]]、[[JCB]] — 信用卡清算
- **跨平台:** [[一卡通]] (iPASS) — 電子支付互通

**中游 (行動支付):**
- **LINE Pay** (第三方行動支付、信用卡綁定收款、跨平台電子支付服務)

**下游 (商家與消費者):**
- **大型零售:** 百貨、量販店、連鎖餐飲
- **中小商家:** 便利商店、飲料店、夜市攤商
- **線上平台:** 電商網站與外送平台

## 主要客戶及供應商
### 主要客戶
- **大型零售:** 百貨商場、量販店、連鎖餐飲 — 行動支付收款
- **中小商家:** 便利商店、飲料店、夜市攤商 — QR Code 收款
- **線上電商:** 電商網站與外送平台 — 線上支付串接

### 主要供應商
- **母集團:** [[LINE]] — 通訊平台與用戶基礎
- **收單銀行:** [[中國信託]]、[[國泰世華銀行]] — 信用卡收單通道
- **信用卡組織:** [[VISA]]、[[Mastercard]]、[[JCB]] — 國際清算
- **跨平台:** [[一卡通]] (iPASS) — 電子支付互通
