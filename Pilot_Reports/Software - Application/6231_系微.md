# 6231_系微

## 業務簡介
**板塊:** Technology
**產業:** Software - Application
**市值:** 12,235 百萬台幣
**企業價值:** 11,379 百萬台幣

系微股份有限公司 (6231，Insyde Software) 是一家全球領先的系統韌體(Firmware)與軟體工程服務供應商。公司的核心產品為UEFI系統韌體(如InsydeH2O BIOS)以及基板管理控制器(BMC)韌體(如Supervyse)。系微的韌體是銜接電腦硬體與作業系統的關鍵橋樑，廣泛應用於個人電腦、[[伺服器]]、[[物聯網]]及邊緣運算設備。近年來，受惠於[[AI]] PC浪潮(如[[Intel]]/[[AMD]]新晶片及高通Snapdragon X平台)與[[AI[[伺服器]]]]的強勁需求，系微的BIOS授權金及NRE(委託設計)收入穩健增長，在[[AI]] PC市場擁有極高市佔率。

## 供應鏈位置
*   **上游**: [[Intel]]、[[AMD]]、[[NVIDIA]]、高通(Qualcomm)、ARM及Aspeed(信驊)等全球頂尖晶片設計原廠。
*   **中游**: **系微** (UEFI BIOS系統韌體、BMC[[伺服器]]管理韌體之設計開發、客製化修改及授權服務)。
*   **下游**: 全球各大電腦及[[伺服器]]品牌廠([[OEM]])、專業電子代工廠([[ODM]])及雲端[[資料中心]]業者。

## 主要客戶及供應商
### 主要客戶
*   [[HP]]、[[Dell]]等PC品牌，浪潮/新華三等中國[[伺服器]]廠，以及[[廣達]]、[[緯創]]等台灣大型[[ODM]]廠。

### 主要供應商
*   微軟(Windows)、[[Google]](Android)及開源社群(如OpenBMC)，提供軟體作業環境與底層架構。
