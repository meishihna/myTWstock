# 8284 - 三竹

## 業務簡介
**板塊:** Technology
**產業:** Software - Application
**市值:** 3,025 百萬台幣
**企業價值:** 1,818 百萬台幣

三竹資訊 (8284，Mitake) 為台灣金融資訊服務領導廠商，成立於 1991 年。公司提供行動看盤下單系統、行動銀行系統、企業簡訊平台、推播系統及企業即時通訊 (EIM) 等解決方案。國內行動看盤下單現有客戶涵蓋 [[元大證券]]、[[富邦證券]]等 46 家證券商，國外股票看盤下單客戶包括 [[凱基證券]]、[[富邦證券]]等 19 家。三竹亦與 [[Sony]]、[[SAMPO]]、[[Panasonic]] 等大廠建立長期策略夥伴關係，推出智慧電視影音看盤服務。公司為簡訊業界唯一獲行政院頒發電信防堵感謝狀之企業。

## 供應鏈位置
**上游 (資訊源與平台):**
- **行情資訊:** [[臺灣證券交易所]]、[[櫃買中心]] (即時行情資料)
- **電信通道:** 各大電信營運商 (簡訊發送通道)
- **雲端平台:** [[AWS]]、[[Microsoft]] Azure

**中游 (金融資訊服務):**
- **行動金融:** **三竹資訊** (行動看盤下單/行動銀行/企業簡訊/推播/EIM)

**下游 (終端客戶):**
- **證券業:** [[元大證券]]、[[富邦證券]]、[[凱基證券]]等 46 家國內券商
- **銀行業:** 各大商業銀行 (行動銀行系統)
- **家電合作:** [[Sony]]、[[Panasonic]]、[[SAMPO]] (智慧電視看盤)
- **企業簡訊:** 各大企業與政府機關 (簡訊推播服務)
