# 5201 - 凱衛

## 業務簡介
**板塊:** Technology
**產業:** Software - Application
**市值:** 1,168 百萬台幣
**企業價值:** 844 百萬台幣

凱衛資訊 (Kway) 為台灣老牌 **金融軟體服務商**，深耕證券期貨領域。
業務核心：
1.  **金融中後台系統**: 證券/期貨核心帳務系統、結算系統。
2.  **交易前端工具**: 獨家代理 **MultiCharts** (程式交易軟體) 於台灣之銷售，擁有廣大程式交易客群。
3.  **系統整合**: 代理銷售國外主機與軟體 (如 UNIX [[伺服器]], Fix Engine)。
4.  **FinTech**: 推出「籌碼大股東」、「高股息存股」等理財 App。
市場地位：在外資券商在台據點之市佔率極高 (約七成)。

## 供應鏈位置
*   **中游 (本體)**: **凱衛 (5201)** - 金融軟體開發 & 系統整合 (SI).
*   **上游**: 軟硬體原廠 (Broadridge, MultiCharts 原廠, 主機廠), 交易所數據源 ([[台灣證券交易所]]).
*   **下游**:
    *   **金融機構**: 證券商 (元大, 凱基, 富邦, 外資券商), 期貨商.
    *   **散戶/專業投資人**: MultiCharts 使用者, 理財 App 用戶.

## 主要客戶及供應商
### 主要客戶
*   **外資券商**: [[Goldman Sachs]], [[Morgan Stanley]], [[UBS]], [[JP Morgan]] (台灣分公司).
*   **本土券商**: [[元大金]], [[凱基證券]], [[富邦金]].

### 主要供應商
*   **軟體原廠**: Broadridge (Fix Engine), MultiCharts (Trading Platform).
