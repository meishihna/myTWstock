# 8455 - 大拓-KY

## 業務簡介
**板塊:** Technology
**產業:** Computer Hardware
**市值:** 565 百萬台幣
**企業價值:** 683 百萬台幣

大拓控股股份有限公司 (8455，大拓-KY，[[Daito Me Holdings]]) 前身成立於 1957 年，註冊地為開曼群島，主要在日本及台灣從事電子零件及機械設備之加工與銷售。公司提供嵌入式產品 (工業電腦解決方案、嵌入式主機板、觸控面板電腦、周邊元件)、控制面板、線圈、機器手臂系統，以及 CPU 主機板、控制設備元件、伺服馬達、電源供應器、連接器等產品。

此外，公司亦從事軟性銅箔電路板 ([[FPC]]) 及黏著膜之製造。產品服務於多元產業之工業自動化設備需求。公司在日本市場深耕多年，具備日系客戶之長期信賴關係。

## 供應鏈位置
**上游 (電子零件與材料):**
- **處理器:** [[Intel]]、[[AMD]] — 嵌入式 CPU 主機板
- **伺服馬達:** 日系伺服馬達原廠
- **連接器/電源:** 工業級連接器及電源供應器
- **FPC 材料:** 軟性銅箔基板及黏著膜原料

**中游 (電子零件加工與嵌入式方案):**
- **嵌入式系統:** **大拓** (工業電腦解決方案、嵌入式主機板)
- **自動化設備:** **大拓** (控制面板、線圈、機器手臂系統)
- **FPC 製造:** **大拓** (軟性銅箔電路板及黏著膜)

**下游 (日本工業自動化市場):**
- **工業自動化:** 日本工業自動化設備製造商
- **電子製造:** 日本及台灣電子產品製造商
- **設備代理:** 日本工廠自動化設備代理通路

## 主要客戶及供應商
### 主要客戶
- **工業自動化:** 日本工業自動化設備製造商 — 控制面板/機器手臂
- **電子製造:** 日本及台灣電子產品製造商 — 嵌入式主機板/零件
- **設備代理:** 日本工廠自動化設備代理通路

### 主要供應商
- **處理器:** [[Intel]]、[[AMD]] — 嵌入式 CPU 主機板
- **伺服馬達:** 日系伺服馬達原廠 (如 [[Yaskawa]]、[[Mitsubishi Electric]])
- **連接器:** 國際工業級連接器供應商
- **FPC 材料:** 軟性銅箔基板原料供應商
