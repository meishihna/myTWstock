# 2414 - 精技

## 業務簡介
**板塊:** Technology
**產業:** Computer Hardware
**市值:** 7,011 百萬台幣
**企業價值:** 8,486 百萬台幣

精技電腦 (2414，[[Unitech Computer]]) 成立於 1979 年，總部位於台北內湖，為台灣最早投入資訊產品通路之企業，深耕台灣 IT 市場逾 45 年。公司經營兩大核心業務：資訊科技產品通路及自動辨識資料收集 ([[AIDC]])。

通路事業以精技電腦為主體，代理經銷各大國際 IT 品牌之電腦、周邊、伺服器、網路設備、印表機、數位看板及企業解決方案。AIDC 事業則以子公司[[精聯電子]] (3652) 為核心，研發製造軍工規 PDA 式行動電腦、工規手持式行動電腦、條碼掃描器、[[RFID]] 讀取器及門禁差勤終端機。

## 供應鏈位置
**上游 (國際 IT 品牌原廠):**
- **電腦品牌:** [[ASUS]]、[[Acer]]、[[Apple]] — 桌機/筆電/伺服器代理
- **印表機:** [[HP]]、[[Epson]] — 印表機及耗材代理
- **網路設備:** 國際網通品牌 — 企業級網路設備
- **AIDC 元件:** 條碼掃描引擎、[[RFID]] 讀寫模組

**中游 (IT 通路與 AIDC 研發製造):**
- **IT 通路:** **精技** (台灣 IT 產品通路經銷商)
- **AIDC 製造:** [[精聯電子]] (3652，工規行動電腦/條碼掃描器/RFID 讀取器)
- **企業方案:** **精技** (企業 IT 解決方案整合)

**下游 (企業客戶與政府機關):**
- **企業用戶:** 各產業企業 — IT 設備採購
- **政府機關:** 政府及國營事業 — IT 標案
- **物流倉儲:** 物流/倉儲/製造業 — AIDC 自動辨識方案
- **零售業:** 零售通路 — POS 及條碼掃描設備
